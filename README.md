# Options & Volatility Analysis

This project analyzes options chain data to understand market sentiment and volatility patterns using quantitative techniques.

## Objectives
- Analyze call and put option activity
- Calculate Put-Call Ratio (PCR)
- Study implied volatility trends
- Generate sentiment indicators from options data

## Data Source
- Options chain data sourced from Yahoo Finance

## Key Metrics
- Put-Call Ratio
- Open Interest (Calls vs Puts)
- Implied Volatility (basic analysis)

## Tech Stack
- Python
- Pandas
- NumPy
- yfinance

## Workflow
1. Fetch options chain data
2. Clean and structure call and put datasets
3. Compute market sentiment indicators
4. Analyze volatility patterns

## Use Case
Designed to support quantitative research and market sentiment analysis in trading environments.
